Help: Searching with open semantic search

Search operators and queries:
  • corrupt* will find corrupt, corruption, corrupted ...
  • AND: search engine will find all documents which contains the words engine AND search
  • OR: search OR research will find all documents which contains the word search or the word research
  • Combine criterias: (search OR research) AND (engine OR software)
  • Phrase search: "search engine" will find exactly this phrase and will not find i search an engine
  • Exclude: search -engine will find all documents which contain the word search and do not contain the word engine
  • Fuzzy search: similar~ words~ would find similer works, because both words are set to fuzzy search. Usefull if you want to consider typos in the documents or if some documents are images or scanned documents (like pdf-documents containing scanned pages only in graphical formats instead of digital text), so you maybe find more documents for your query even if the results of automated text-recognition (ocr) are not perfect)
